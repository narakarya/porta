# kamal has moved

The Kamal Deploy extension that lived at `extensions-bundled/kamal/` is now developed in its own repo:

**[narakarya/porta-kamal](https://github.com/narakarya/porta-kamal)**

## Install / reinstall

In Porta → Settings → Extensions → "Install from GitHub":

```
narakarya/porta-kamal
```

Pin a release:

```
narakarya/porta-kamal@v0.1.0
```
