# git-manager has moved

The Git Manager extension that lived at `extensions-bundled/git-manager/` is now developed in its own repo:

**[narakarya/porta-git-manager](https://github.com/narakarya/porta-git-manager)**

## Install / reinstall

In Porta → Settings → Extensions → "Install from GitHub":

```
narakarya/porta-git-manager
```

Pin a release:

```
narakarya/porta-git-manager@v0.7.0
```

## If you previously installed via the old URL

The old install URL (`narakarya/porta:extensions-bundled/git-manager`) is no longer resolvable — the "Update" button in Settings will error against it.

Click the trash icon next to the entry, then reinstall via the new URL above. Your existing extension data is unaffected (the symlink target / cache is replaced in place).
